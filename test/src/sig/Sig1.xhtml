Sig Test #1




This test sets up <sig:action> element that has inline XMLEvents attributes on it that point to a handler block that has multiple action elements under it.


This XHTML button element has an XMLEvents 'ev:event' attribute value of 'DOMClick' and an 'ev:handler' attribute value of '#fooClicker', meaning that DOMClick signals coming from the button will cause the 'fooClicker' handler block to be run.
Note here the TIBET 'sig:action' action block element that enclose 2 script block elements.
