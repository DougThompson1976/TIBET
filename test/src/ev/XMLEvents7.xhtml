XML Events Test #7




This test sets up an XMLEvents <ev:script> with TIBET Shell which activates on X.


This <ev:script> element has an 'event' value of 'TP.sig.DOM_X_Up' (the TIBET way of observing the 'X' - shift x - key going 'up') and an 'observer' containing '#document', meaning that the document should be the observer of those events. It will alert a message.


This <ev:script> element has an 'event' value of 'TP.sig.DOM_Y_Up' (the TIBET way of observing the 'Y' - shift y - key going 'up') and an 'observer' containing '#document', meaning that the document should be the observer of those events. It will alert the 'JSON source' representation of the signal that triggered the script.


This <ev:script> element has an 'event' value of 'TP.sig.DOM_Z_Up' (the TIBET way of observing the 'Z' - shift z - key going 'up') and an 'observer' containing '#document', meaning that the document should be the observer of those events. It will alert the 'JSON source' representation of the target of the signal that triggered the script.
