Bind Repeat JSON Paging




This test sets up a 'repeat' binding between the group of fields and a piece of JSON data, set up by the <tibet:data> tag embedded in this document. The repeating fields and bindings are automatically generated from the data, which is a collection.


This test also uses both a 'bind:repeatsize' and a 'bind:repeatindex'. These allow control over how many repeat items are showing and what the 'starting point' within the data to begin showing data is.
