Getting started with TIBET™ is easy!
Three quick steps and you'll be running. Ok. Four.
TIBET leverages Node.js for its command line tooling so be sure you have it installed first. (Get Node.js here).
You don't need to run a Node.js server for your apps.
Install =>
npm install -g tibet
or
git clone https://github.com/TechnicalPursuit/TIBET.git;
cd TIBET; npm install -g .
Clone =>
tibet clone appname [--dna template]
Start =>
cd appname; tibet start
You'll find extensive conceptual and design documentation on the wiki.
The 'tibet help' and 'tibet man' commands provide API documentation.
