Getting started with TIBET™ is easy!
Three quick steps and you'll be running.
Ok, four. You can run any server, or zero server, but the 'tibet' command relies on Node.js [get it here].
With that foundation in place you're almost there.
Install =>
npm install -g tibet
Clone =>
tibet clone appname [--dna template]
Start =>
cd appname
tibet start
open index.html
You'll find extensive conceptual and design documentation on the wiki.
The 'tibet help' and 'tibet man' commands provide API documentation.
